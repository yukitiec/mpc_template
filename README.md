> cmake build and compile. If osqp include and lib directories are not recognized correctly, \\ set explicitly instead of find_package

> ./bootstrap-vcpkg.sh \\
> ./vcpkg integrate install \\
> ./vcpkg install nlopt \\
- [Catch2](https://github.com/catchorg/Catch2)
